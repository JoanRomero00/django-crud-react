## Comandos de ayuda
Crear un proyecto
```bash
django-admin startproject mysite .
```
Crear una app
```bash
django-admin startapp myapp
```
Actualizar DB
```bash
python manage.py makemigrations
```
```bash
python manage.py migrate
```
Crear SuperUsuario
```bash
ython manage.py createsuperuser
```
